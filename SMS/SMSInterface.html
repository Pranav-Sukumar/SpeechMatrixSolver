Speech Matrix Solver

A web interface speech-based matrix solver built by Ashwat C., Srisai N., Pranav S., and Seenu M. for EECS 16A in Fall 2019.

Sorry, your Browser Doesn't Support the Web Speech API. Try Opening This Demo In Google Chrome.


To use this interface, follow the following steps:

  1. Click "Start Recognition" when you are ready to begin.
  2. Verbally dictate "Solve a [M] by [N] matrix with values of [Value 1] dot [Value 2] dot" until all values have been spoken.
  3. When finished with reading the values of the matrix, wait a few seconds for the speech detection to automatically stop.
  4. Verify the input was recorded accurately and edit mistakes if necessary.
  5. Finally click "Solve Matrix!" to see the solution below!
